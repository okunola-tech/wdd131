FitLife Tracker – Website Plan

Site Name

FitLife Tracker

This name was chosen because it clearly represents a fitness-focused website that helps users track workouts, goals, and healthy habits in everyday life.

Optional domain: fitlifetracker.com

Site Purpose

The purpose of the FitLife Tracker website is to provide users with a simple, easy-to-use platform where they can track workouts, view fitness tips, and stay motivated toward healthier living.

Scenarios

  • How can I track my weekly workouts and fitness progress?
  • Where can I find beginner-friendly fitness tips and routines?

Color Scheme

The following colors will be used throughout the website:

#2b7a78 – Headings, navigation, accents
#def2f1 – Background color
#17252a – Body text and footer

Typography

Roboto will be used throughout the site.

  • Headings: Roboto Bold
  • Body text: Roboto Regular
